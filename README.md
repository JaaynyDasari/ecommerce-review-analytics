# E-Commerce Review Analytics

This project analyzes large-scale e-commerce review data using PySpark to extract insights on customer sentiment and product performance.

## Features
- Data cleaning and preprocessing of large review datasets (~800K records)
- Exploratory Data Analysis (EDA) to identify trends in ratings and reviews
- Sentiment analysis using VADER to classify reviews as positive, negative, or neutral
- Analysis of product performance based on review count, ratings, and sentiment

## Tech Stack
- Python
- PySpark
- NLP (VADER Sentiment Analysis)
- Pandas, NumPy

## Objective
To understand customer behavior and sentiment patterns from large-scale review data and derive actionable insights.
